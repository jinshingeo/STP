# 프로젝트 CLAUDE.md — 성동구 STP 논문 분석

## 연구 개요
- **주제**: 열환경을 반영한 보행 Space-Time Prism(STP) 분석
- **연구 질문**: 폭염 시간대에 UTCI 기반 열환경을 소프트 제약으로 적용했을 때 보행 PPA가 얼마나 감소하는가
- **비교 지역**: 응봉동(열취약, 교량 노출) vs 성수동(열완충, 서울숲 인접)
- **핵심 지표**: PPA 감소율(%) = (Classic PPA − Thermal PPA) / Classic PPA × 100

## 폴더 구조

## 분석 환경
- Python 기반 분석 (Jupyter Notebook)
- 주요 라이브러리: geopandas, networkx, osmnx, shapely, matplotlib 등
- 좌표계: EPSG:5186 (서울 지역 분석 기준)
